### 클래스명과 객체 참조 변수명

```java
Penguin pororo = new Pengin();
```
*팽귄 한 마리가 태어나니 팽귄 역할을 하는 pororo라 이름 지었다.*

```java
pororo.name = "뽀로로";
```
*pororo의 name을 "뽀로로"라 하자.*

```java
pororo.habitat = "남극";
```
*pororo의 habitat(서식지)를 "남극"이라 하자.*

```java
pororo.ShowName();
```
*pororo야 너의 이름을 보여줘.*

```java
pororo.showHabitat();
```
*pororo야 너의 서식지를 보여줘.*
