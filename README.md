## 책 정보
- **제목**: 스프링 입문을 위한 자바 객체 지향의 원리와 이해
- **저자**: 김종민
- **출판사**: 위키북스

## 학습 목표
- 자바 객체 지향 프로그래밍의 핵심 개념 이해
- SOLID 원칙 습득 및 실습
- 디자인 패턴의 기초 학습
- 스프링 프레임워크와 객체 지향의 연관성 파악
- AOP(관점 지향 프로그래밍) 개념 이해
